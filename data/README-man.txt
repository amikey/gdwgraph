MAN: do you want to be a real MAN?

Goal: kill all movable objects (men, big bosses, butterflies)
There are 10 levels of the game by default,


Control keys:

In intro:
ESC - quit to OS
Space - start game
1,2 - change number of human players

In game:
1 player: ARROWS to move, SPACE: fire, ENTER: change weapon: arrow, fork, axe
2 player: Q,E,W,A to move, D: fire, S: change weapon
ESC: quit game

F1: pause
F5: double/normal mode (for one player game only)
F10: fast quit to OS

Shift+TAB: skip to next level


== ======= ==
== RUSSIAN ==
========== ==

Цель: уничтожить все движущиеся объекты (человечков, больших боссов, бабочек)
Всего 10 уровней для прохождения.


Клавиши управления:

Заставка:
ESC - выход в ОС
Space - старт игры
1,2 - сменить число игроков (оба игрока управляются людьми на одном экране)

Игра:
Кол-во оружия выводится внизу экрана рядом с каждым человеком

1 игрок: СТРЕЛКИ перемещение, ПРОБЕЛ: огонь, ENTER: сменить оружие: стрелы, вилы, топоры
2 игрок: Q,E,W,A перемещение, D: огонь, S: сменить оружие

ESC: конец игры

F1: пауза
F5: режим двойной/нормальный (только для режима игры одного игрока)
F10: быстрый выход в ОС, минуя заставку

Shift+TAB: прыгнуть на следующий уровень
